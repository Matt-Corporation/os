# matt Corp. Code of Conduct

> Note: [we are not lawyers](WE_ARE_NOT_LAWYERS.md).

By using, modifying, forking or in any other way interacting with this software, you agree to its License and Code of Conduct.
The Code of Conduct applies for all contributions made to this software by all contributors.
If you're a maintainer of this software, you agree to strictly enforce the Code of Conduct.
Please note that we might edit or revise this document in the future.

## Rules

The Code of Conduct consists of so-called "rules", which are a set of good practices when contributing or working with matt Corp. software or products.

### 1. The use of AI

The use of any AI generated content must be clearly noted. This type of content is **not** typically allowed.
The same applies for generative AI features; do **not** implement them into any matt Corp. software or products unless it is genuinely useful, not gimmicky, and doesn't burn forests and doesn't waste water.
