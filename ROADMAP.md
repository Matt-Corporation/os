# Roadmap

> Taken from [here](https://osdev.wiki/wiki/What_Order_Should_I_Make_Things_In%3F).

- [x] Printing strings and integer numbers (both decimal and hex) on the screen
- [x] Outputting to a serial port
- [x] Having a working and reliable interrupt/exception handling system that can dump the contents of the registers (and perhaps the address of the fault) will be very useful.
- [ ] Plan your memory map (virtual, and physical): decide where you want the data to be
- [ ] The heap: allocating memory at runtime (malloc and free) is almost impossible to go without. It should be implemented as soon as possible.
